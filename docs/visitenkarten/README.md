# FIECON Visitenkarten

Professionelle Visitenkarten im Corporate Design der Webseite.

## Dateien

- **visitenkarte.html** – Druckfertiges Layout (helles Design, Beige-Hintergrund)

## Verwendung

1. **HTML-Datei im Browser öffnen** (Doppelklick oder `open docs/visitenkarten/visitenkarte.html`)
2. **Drucken** mit `Strg+P` (Windows) bzw. `Cmd+P` (Mac)
3. **Als PDF speichern**: Im Druckdialog „Als PDF speichern“ wählen
4. **Hintergrundgrafiken aktivieren**, damit Logo und Farben gedruckt werden

## Druckformat

- **Kartenformat:** 85 × 55 mm (Europäisches Standardformat)
- **Blatt:** A4, 10 Karten pro Seite (2 Spalten × 5 Zeilen)
- **Schneidemarken:** Karten mit feinem Rahmen für präzises Schneiden

## Für die Druckerei

Die HTML-Datei kann direkt als PDF exportiert und an eine Druckerei gesendet werden. Alternativ die PDF-Datei mit „Als PDF speichern“ erzeugen und mit **3 mm Beschnitt** angeben, falls die Druckerei Beschnitt benötigt.
